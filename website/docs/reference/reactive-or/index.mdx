# reactiveOr

Alternative: `or$$`

### Types

```ts
function reactiveOr(
  a: IObservable<boolean>,
  b: IObservable<boolean>,
): IObservable<boolean>
```

### Definition

Creates an Observable which performs the boolean `OR` operation (`||`) between the values sent by two Observables, and emits the result.

### Diagram

```mermaid
sequenceDiagram

participant A as observableA
participant B as observableB
participant reactiveOr as reactiveOr(observableA, observableB)
participant OUT

A->>reactiveOr: false
B->>reactiveOr: false
reactiveOr->>OUT: false
B->>reactiveOr: true
reactiveOr->>OUT: true
```

### Example

#### Perform the boolean "OR" of two Observables

```ts
const subscribe = reactiveOr(
  single(true),
  single(false),
);

subscribe((value: boolean) => {
  console.log(value);
});
```

Output:

```text
true
```
