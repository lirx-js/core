# ofWithNotifications

Alternative: `ofN`

### Types

```ts
function ofWithNotifications<GValue>(
  ...values: GValue[]
): IObservable<IOfObservableNotifications<GValue>>
```

```ts
type IOfObservableNotifications<GValue> = IFromArrayObservableNotifications<GValue>;
```

### Definition

Creates an Observable from a list of values. It emits the values one by one in the form of `next` Notifications
and then complete (`complete` Notification).

It's identical of [fromArrayWithNotifications](/docs/reference/from-array-with-notifications/) but use spread values instead.

### Example

#### Emit the sequence of values: 0, 1, 2, 3

```ts
const subscribe = ofWithNotifications(0, 1, 2, 3);

subscribe((notification) => {
  console.log(notification.name, ':', notification.value);
});
```

Output:

```text
next: 0
next: 1
next: 2
next: 3
complete
```
