# INextNotification

### Types

```ts
type INextNotification<GValue> = INotification<'next', GValue>;
```

### Definition

Used to emit a value. This is used as a replacement of the `next` *event* of RxJS.

You can create a *NextNotification* with the function [createNextNotification](/docs/reference/create-next-notification/).
