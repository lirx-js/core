# createMulticastSource

### Types

```ts
function createMulticastSource<GValue>(): IMulticastSource<GValue>
```

### Definition

Creates a [IMulticastSource](/docs/reference/multicast-source/).


### Example

#### Subscribe many times to the same Source

```ts
const source = createMulticastSource<number>();

source.subscribe((value: number) => {
  console.log('value - A:', value);
});

source.subscribe((value: number) => {
  console.log('value - B:', value);
});

source.emit(1);
source.emit(2);
```

Output:

```text
value - A: 1
value - B: 1
value - A: 2
value - B: 2
```

##### RxJS equivalent

```ts
const source = new Subject<number>();

source.subscribe((value: number) => {
  console.log('value - A:', value);
});

source.subscribe((value: number) => {
  console.log('value - B:', value);
});

source.next(1);
source.next(2);
```
